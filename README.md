# Wallet Message Pay

Replaced the @/components/ui/input-otp import (which is not available) in OTPVerification with a self-contained 6-box numeric OTP input built from native inputs, including paste distribution, backspace/arrow navigation, auto-advance and auto-submit. No new dependencies. The phone library was already self-contained (no libphonenumber-js import remains), so the landing page now renders real content.
